About This Site

We've tried to make a site that looks just good enough without looking like we spent a bunch of time or money on it. It needs to show that we're technically competent. At the same time, we are not a website development company. We are interested in getting stuff done. We are an engineering company first and foremost, and that means form follows function.

This site was built based on the Kanzi theme available from WrapBootstrap for $16. Don't build when you can buy. All site code is stored in a git repository on BitBucket, which automatically publishes site changes to Windows Azure. Server Side Includes scripting keeps the header and footer consistent across all pages. Site stats are compiled by Google Analytics. The blog is a Ghost blog hosted on Digital Ocean. Total site setup and content was done nights/weekends over the course of about 3 weeks.
